allô (interj., téléph.)	Zeggen bij opnemen of het gesprek starten. In NL vaak: *Hallo* / *met …*.							
J’écoute (form. téléph.) / Ici (loc.)	*Ici* = hier (zaak) — *Ici, le bistro* = (u spreekt met) ons, het bistro. *J’écoute* = ik luister, wat is uw wens? — typisch, bedrijf.							
réserver (v.); une réservation (f.)	reserveren; (een) reservering. *Je voudrais réserver* = later plus volledig, hier: *une table* + *pour deux* (formule).							
une table pour deux	een tafel voor twee. *Ce soir* / *le soir* / *samedi soir* = (van)avond, zaterdagavond, later preciseren.							
À quelle heure ? (interr.) / à + heure	Hoe laat? *À vingt heures* = om twintig uur, *à 20h* = idem, schrift, oral.							
c'est noté (formule)	staat erin / genoteerd, begrepen, OK (restaurant, rendez-vous) — *noter* = noteren, schrijven.							
au nom de … (formule)	op naam (van) … — *Réservé au nom de Martin* = gereserveerd op naam van Martin. A1, par patronyme.							
deux personnes (nombr., simple) / le nom (m., patronyme)	twee personen; *Mon nom* = mijn (familie)naam. *Avec* + letter = de letter herhalen op de telefoon, praktisch (A1).							
que désirez-vous ? (f., au téléph., poli) / bonne soirée (f.)	Wat wenst u? (dienst) — in het NL vaak: *Wat kan ik voor u doen?*; *Bonne soirée* = fijne avond (bij afschied).
