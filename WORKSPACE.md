# PSTUDY Workspace

**Run and edit from:** `C:\pstudy-web`

This folder is the primary PSTUDY project. Use it for:
- `npm run dev` — local development
- All code edits
- `.env.local` — Supabase and Google Cloud credentials

A backup/copy may exist at `y:\PSTUDY\Cursor\pstudy-web\`, but changes there will not appear when running from C drive. Keep C as the single source of truth.
